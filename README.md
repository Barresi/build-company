# Build Company

Проект системы управления строительной компанией, реализованный на двух языках программирования.

## Структура репозитория

```
├── cpp/          # C++ реализация проекта (в разработке)
└── csharp/       # C# реализация проекта (в разработке)
```

## C++ версия

Реализация системы с упором на управление памятью и низкоуровневые концепции C++.

**ООП в стиле C++:**
- Абстрактные базовые классы с чисто виртуальными функциями
- Разделение интерфейса (.hpp) и реализации (.cpp)
- Строгая типизация через `enum class`
- Inline функции для вспомогательных преобразований

**Структура файлов:**
```
cpp/build-company/
├── *.hpp          # Заголовочные файлы с объявлениями классов
├── *.cpp          # Файлы реализации
└── main.cpp       # Точка входа с демонстрацией всех возможностей
```


## C# версия

Реализация системы с использованием современных возможностей C# и автоматического управления памятью.

**ООП в стиле C#:**
- Абстрактные базовые классы (`abstract class`)
- Инкапсуляция через свойства и методы
- Строгая типизация через `enum` с методами расширения
- Автоматическое управление памятью (Garbage Collector)

**Структура файлов:**
```
csharp/build-company/
├── *.cs                # Файлы классов с полной реализацией
├── Enums.cs            # Перечисления и вспомогательные методы
└── Program.cs          # Точка входа с демонстрацией всех возможностей
```
