# Build Company

Проект системы управления строительной компанией, реализованный на двух языках программирования.

## Структура репозитория

```
├── cpp/          # C++ реализация проекта (в разработке)
└── csharp/       # C# реализация проекта (в разработке)
```

## C++ версия

Реализация системы с упором на управление памятью и низкоуровневые концепции C++.

**ООП в стиле C++:**
- Абстрактные базовые классы с чисто виртуальными функциями
- Разделение интерфейса (.hpp) и реализации (.cpp)
- Строгая типизация через `enum class`
- Inline функции для вспомогательных преобразований

**Структура файлов:**
```
cpp/build-company/
├── *.hpp          # Заголовочные файлы с объявлениями классов
├── *.cpp          # Файлы реализации
└── main.cpp       # Точка входа с демонстрацией всех возможностей
```
